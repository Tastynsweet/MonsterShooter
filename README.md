# MonsterShooter

A completed game project focused on core gameplay, mechanics, and presentation.  
The final remaining task is adding sound effects and audio polish.

## Overview
Defend the Depths is a 2D game where the player controls cannons and defend their ship from monsters. The project was developed to demostrate game design, programming, system implementation and drawing skills. Some AI has been used for learning purposes and code polish.

The game is fully playable from start to finish, with all major mechanics implemented.

## Features
- Complete gameplay loop
- Player controls and interactions
- Enemy behavior
- UI and scoring system
- Visual effects and animations

## Current Status
**Initial Content complete.**
The game is fully playable and feature-complete
The only remaining work is:
- Sound effects
- Background music
- Audio balancing and polish
- Post processing

## Controls
- <- / -> = Left and right arrows to control cannon rotation
- Spacebar = Shoot
- Q/W/E = Switch between cannons
(More on the controls in the game menu)
